Diagrama Visual: Conceito de Backtracking

Explicação do Diagrama

  • Nó Raiz (Início): Representa o estado inicial do problema.
  • Linha Verde: Mostra o caminho de exploração que o algoritmo está seguindo.
  • Nó com X vermelho: Representa um beco sem saída - uma solução parcial inválida.
  • Seta Vermelha Pontilhada: Indica o backtrack - o algoritmo volta para o nó pai anterior.
  • Linha Azul: Mostra o novo galho escolhido após o backtrack.
  • Nó com ✓ verde: Representa uma solução válida encontrada.
